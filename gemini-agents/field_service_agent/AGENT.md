# Field Service Agent

## Purpose

Helps technicians document completed plumbing jobs quickly and accurately.

## What It Does

**Collects job information:**
- Customer name and location
- Work performed and hours spent
- Materials used

**Validates data:**
- Looks up customer in database
- Checks for duplicate invoices
- Confirms details with technician

**Prepares for billing:**
- Creates complete job data package
- Hands off to office agent for billing processing

## Goal

Minimize administrative work for busy technicians while ensuring accurate, complete job documentation for billing.
